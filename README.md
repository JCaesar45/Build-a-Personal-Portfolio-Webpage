# Personal Portfolio Webpage

This is a simple, responsive personal portfolio webpage built with HTML and CSS.  
It demonstrates basic web development skills and meets the requirements for a portfolio project.

---

## Demo

View the live demo here: *(Add your live URL if deployed)*

## Features

- Fixed navigation bar with links to different page sections
- Full viewport welcome section with a heading
- Projects section showcasing project tiles with links
- Responsive design with media queries for smaller screens
- External profile link (GitHub) opening in a new tab

---

## Project Structure

``

/index.html
/styles.css
/README.md

``

- `index.html`: Contains the markup for the portfolio including navbar, welcome section, and projects section.
- `styles.css`: Stylesheet with layout, colors, and responsiveness.
- `README.md`: Project documentation.

---

## User Stories Fulfilled

1. Portfolio includes a welcome section with the id `welcome-section`.
2. The welcome section contains a non-empty `<h1>` element.
3. Portfolio includes a projects section with the id `projects`.
4. At least one project tile with the class `project-tile` is present.
5. Each project tile includes a link to an external project.
6. A navbar with the id `navbar` is always fixed at the top.
7. Navbar contains links that navigate to different sections on the page.
8. There is a profile link (`id="profile-link"`) that opens the GitHub profile in a new tab.
9. The site uses a media query to improve mobile responsiveness.
10. The welcome section height matches the viewport height.

---

## How to Use

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Customize the project links and profile URL in the HTML file.
4. Modify styles in `styles.css` to fit your personal branding.

---

- CSS3 (Flexbox, Media Queries)
